# Node JS REST API Template

## 1. 프로젝트 설치하기
```
npm install
```

## 2. 서버 실행하기
```
node server.js
```

## 3. URL 테스트하기
- POST - /login
- POST - /logout
- POST - /signUp
- GET - /home
